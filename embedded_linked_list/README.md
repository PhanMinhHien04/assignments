# Linked List Implementation in C

# 1. Description

This project implements a Singly Linked List in C without using any external libraries or STL containers. The linked list supports inserting nodes at the head and tail, removing the head node, traversing the list, and releasing all allocated memory.

The implementation uses dynamic memory allocation (`malloc()` and `free()`) and demonstrates the fundamental concepts of linked-list data structures commonly used in embedded systems and low-level software development.

# 2. Build
