# Memory Pool Implementation in C

# 1. Description

This project implements a fixed-size Memory Pool in C using a singly linked free list. The pool contains 16 memory blocks, each capable of storing 32 bytes of data. The implementation provides efficient memory allocation and deallocation without relying on dynamic memory functions such as `malloc()` and `free()`.

# 2. Build

Compile the project using:
